# Descripción
Este proyecto fue desarrollado en Python aplicando Programación Orientada a Objetos (POO). El sistema representa de manera básica la gestión de un restaurante mediante clases, objetos y una estructura modular organizada en diferentes archivos.

# Estructura del proyecto
restaurante_app/
│
├── modelos/
│   ├── producto.py
│   └── cliente.py
│
├── servicios/
│   └── restaurante.py
│
└── main.py

# Funcionalidades
- Registro de productos.
- Registro de clientes.
- Gestión del restaurante.
- Visualización de productos y clientes.
- Actualización del precio de un producto.

# Reflexión
La organización modular permite separar responsabilidades entre los diferentes archivos del proyecto, facilitando la lectura, el mantenimiento y la reutilización del código. Además, la Programación Orientada a Objetos ayuda a representar entidades del mundo real mediante clases y objetos, haciendo que el software sea más organizado y fácil de ampliar.
